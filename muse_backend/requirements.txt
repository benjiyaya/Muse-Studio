# ── Step 1: PyTorch with CUDA (REQUIRED — do NOT use plain pip install torch) ─
#
#   CUDA 12.8 (RTX 50xx Blackwell / RTX PRO):
#     pip install torch==2.8.0 torchvision torchaudio --index-url https://download.pytorch.org/whl/cu128
#
#   CUDA 12.1 (RTX 30xx / 40xx Ada):
#     pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
#
# Installing torch from PyPI (without --index-url) gives a CPU-only build.
# The ltx-core package will try to install torch as a dependency — always
# reinstall with the correct CUDA index URL afterwards if that happens.
